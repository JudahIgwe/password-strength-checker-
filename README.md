# Password Strength Checker

A Python program that checks how secure a password is.

## What it does
Takes a password as input and rates it as Weak, Medium or Strong based on:
- Length of 8 or more characters
- Contains uppercase letters
- Contains lowercase letters
- Contains numbers
- Contains special characters

## How to run
1. Install Python 3
2. Run the file: password_checker.py
3. Enter a password when prompted
4. The program will tell you if it is Weak, Medium or Strong

## Built with
- Python 3
- re module (regular expressions)
